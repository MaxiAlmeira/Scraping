- Asegurarme primero de instalar python
- Podemos utilizar la consola de git 
- Corroborar la version de python que tenemos con "python --version"
- Podemos crear el ambiente virtual con el siguiente comando "python3 -m venv env"
- Estando en la carpeta donde creamos el ambiente virtual, lo activamos con "source env/Scripts/activate"
- Para instalar las librerías ponemos "pip install nombreDeLibrería"
- Para desactivar el ambiente virtual ponemos "deactivate"


Si vamos a crear el ambiente virutal a traves de la terminal de VisualStudioCode hacemos lo siguiente:
- "virtualenv nombreDelAmbiente"


Para consultar las librerías que tengo instalada, ya sea en el ambiente virtual o en general, lo hacemos con
- pip list

Para desinstalar librería ponemos: 
- pip unistall nombreLibrería
